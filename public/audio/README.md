# 音频文件说明

此目录用于存放番茄时钟的提示音文件。

## 需要的音频文件

1. **focusEnd.mp3** - 专注时间结束提示音
   - 当专注时间结束, 进入休息时间时播放
   - 建议使用轻柔、舒缓的音效

2. **breakEnd.mp3** - 休息时间结束提示音
   - 当休息时间结束, 进入专注时间时播放
   - 建议使用稍微明显一点的音效

3. **totalCyclesEnd.mp3** - 总循环结束提示音
   - 当所有循环完成时播放
   - 建议使用庆祝性质的音效

## 音频格式要求

- 格式: MP3
- 时长: 建议1-3秒
- 音量: 适中, 不要过于刺耳
